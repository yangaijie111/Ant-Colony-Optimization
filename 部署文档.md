#### 校园食堂就餐路径优化系统 - 部署说明

##### 项目简介

本项目使用**蚁群算法(Ant Colony Optimization, ACO)**优化校园食堂的就餐路径，帮助用户找到从入口到取餐窗口再到就餐区的最优路线，综合考虑行走距离和窗口排队时间。

##### 环境要求

- Python 3.8+
- NumPy

##### 安装步骤

###### 1. 克隆或下载项目

```bash
git clone <项目地址>
cd <项目目录>
```

###### 2. 创建虚拟环境

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/macOS
python3 -m venv venv
source venv/bin/activate
```

###### 3. 安装依赖

```bash
pip install numpy
```

或者使用 requirements.txt（如已创建）：

```bash
pip install -r requirements.txt
```

###### 运行程序

```bash
python main.py
```

###### 输出示例

```
正在优化校园食堂就餐路径...

--- 优化结果 ---
最优路径: [0, 2, 6]
路径节点含义: 0=入口, 1-4=取餐窗口, 5-7=就餐区, 8-9=通道
预估最小总成本 (时间): 45.32 单位
建议选择取餐窗口: B (节点 2)
```

###### 参数配置

可在 `main.py` 中调整以下参数：

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `num_ants` | 10 | 蚂蚁数量 |
| `num_iterations` | 50 | 迭代次数 |
| `alpha` | 1 | 信息素重要程度 |
| `beta` | 2 | 启发函数重要程度 |
| `rho` | 0.5 | 信息素挥发率 |
| `Q` | 100 | 信息素常数 |

##### 节点说明

- **节点 0**: 入口
- **节点 1-4**: 取餐窗口 A/B/C/D
- **节点 5-7**: 就餐区
- **节点 8-9**: 通道节点

##### 注意事项

1. 当前使用随机生成的坐标数据，实际部署时需替换为真实食堂坐标
2. 窗口排队时间为模拟数据，可对接实时监控系统获取真实数据
3. 如需更高精度，可增加 `num_ants` 和 `num_iterations`，但会增加计算时间
